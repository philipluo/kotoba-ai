## 📖 项目简介

**言葉AI (Kotoba AI)** 是一个个人日语学习管理系统，专为多邻国(Duolingo)等日语学习者设计。

通过将每日学习的内容系统化存储、智能分类和个性化复习，帮助学习者建立完整的日语知识体系。

---

## 🎯 核心功能

### 1️⃣ AI辅助录入
- 用户发送截图给AI助手(Opencode)
- AI识别日语内容并返回结构化数据
- 用户在Web系统中预览确认后入库

### 2️⃣ 智能分拣系统
- 自动分词并归类（名词/动词/形容词/助词）
- 动词自动识别原型并生成12种活用形式
- 自动建立50音检索索引

### 3️⃣ 50音图表检索
- 交互式50音图表（平假名/片假名）
- 点击任意发音快速检索相关内容
- 智能匹配（清音→浊音关联）

### 4️⃣ 分类学习表
- 名词表、动词表、形容词表、助词表
- 按词性系统化展示学习内容
- 动词完整活用表（12种变形）

### 5️⃣ 每日一练
- 智能选题算法（新学内容+复习结合）
- 每日20题，权重分配合理
- 一键生成豆包Prompt，支持语音练习

## 🏗️ 技术架构

### 后端
- **Python 3.8+** + **Flask**
- **SQLite** 数据库（单文件，含JSON支持）
- RESTful API设计

### 前端
- **HTML5** + **Vanilla JavaScript**
- **Tailwind CSS** + **DaisyUI** 组件库
- 响应式设计（预留移动端兼容性）

### 数据流
```
用户截图 → AI识别 → Web录入 → 预览确认 → 自动分拣 → 数据库存储
                                              ↓
                                    [50音索引] [动词活用] [分类统计]
                                              ↓
                                    每日一练 ← 分类检索 ← 50音图表
```

## 📝 使用流程

### 日常使用步骤

1. **在多邻国学习后**，截图保存

2. **发送给AI助手**：
   ```
   "请识别这些日语学习内容"
   ```

3. **AI返回结构化数据**：
   ```json
   {
     "original_jp": "私は学生です",
     "hiragana": "わたしはがくせいです",
     "romaji": "watashi wa gakusei desu",
     "meaning": "我是学生"
   }
   ```

4. **在Web系统录入**：
   - 进入「录入页面」
   - 粘贴JSON数据
   - 点击「开始分拣」

5. **预览确认**：
   - 查看自动分词结果
   - 确认/修改动词识别
   - 点击「确认入库」

6. **学习使用**：
   - 浏览「50音图表」检索
   - 查看「分类表」系统复习
   - 每日打开「每日一练」

7. **语音练习**：
   - 生成豆包Prompt
   - 复制到豆包APP
   - 语音对话练习
